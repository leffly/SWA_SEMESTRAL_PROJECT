StoreService->User: 202 ACCEPTED\nRespond with status endpoint\nto poll for the search result
StoreService->Kafka: Post "search_request" message
User->StoreService: GET /store/search/{id}\nGet result of search with id={id}
StoreService->User: 204 NO CONTENT\nSearch result is not ready
Kafka->BookService: Retrieve "search_request" message
BookService->Kafka: Post "search_result" message
Kafka->StoreService: Retrieve "search_result" message
User->StoreService: GET /store/search/{id}\nGet result of search with id={id}
StoreService->User: 302 FOUND\nRespond with the search result
